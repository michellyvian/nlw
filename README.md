## 🖥️ projeto
Esse é um projeto Web responsivo de uma cápsula do tempo para exibir memorias em uma timeline.

## 🚀 tecnologias
Esse projeto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

- HTML;
- CSS;
- Git e GitHub
